matlab编程中遇到的一些小问题-2

记录下matlab编程中遇到的一些编程问题，以防日后再遇到而忘记怎么处理。这次是关于读取图片时遇到的一些问题。

单元数组cell的用法

单元数组可以将多个不同类型的数据和不同维度的数据存储在一起。具体定义用大括号{}，例如下面：

flower_name = {'风信子','桂花','荷花','樱花','洛神花'};

定义了包含五个字符串的单元数组。如果要调用其中某个元素的话，和普通的数组[]不同，需要用flower_name{1}来表示第一个元素，如果使用flower_name(1)，表示的是cell属性的值。

strcat

strcat()函数和其他语言中的函数类似，都是把某几个字符串拼接在一起组成一个新的字符串。例如：

I1 = strcat('a','b','c');

dir

dir()函数可以获得指定文件夹下所有的子文件夹和文件，并存放在一种叫做文件结构体的数组中。例如：

files = dir('D:/new_flowers/','*.jpg'));

上面的代码描述了获取D:/new_flowers中的所有jpg格式的图片，并将其存在文件结构体数组中。

总结

将上面三种函数结合起来看一个例子：

ObjDir2 = 'D:/new_flowers/';%目标图片文件夹路径
flower_name = {'风信子/','桂花/','荷花/','樱花/','洛神花/'};
files = dir(strcat(ObjDir2,flower_name{1},'*.jpg'));
for i = 1:10
    I1 = strcat(ObjDir2,flower_name{1},files(i).name);
    I = imread(I1);%文件所在路径
end

上面描述了获取D:/new_flowers中的第一个文件夹风信子种所有jpg格式的图片，并读取前十张图片。
